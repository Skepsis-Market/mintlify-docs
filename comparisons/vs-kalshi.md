# Skepsis vs Kalshi

How Skepsis compares to the regulated US prediction market.

---

## Quick Comparison

| Feature | Kalshi | Skepsis |
|---------|--------|---------|
| **Regulation** | CFTC regulated (US) | Decentralized (global) |
| **Market type** | Binary (mostly) | Continuous (ranges) |
| **Access** | US residents only | Global |
| **KYC required** | Yes | No (wallet only) |
| **Custody** | Centralized | Self-custody |
| **Blockchain** | None (centralized) | Sui |

---

## Regulation & Access

### Kalshi: Regulated, Restricted

```
✅ CFTC regulated (legal clarity in US)
✅ FDIC-insured deposits
❌ US residents only
❌ Full KYC required
❌ Some markets restricted
❌ Withdrawal limits for new users
```

### Skepsis: Decentralized, Global

```
✅ Accessible globally
✅ No KYC (wallet-based)
✅ Self-custody of funds
✅ No withdrawal limits
❌ Regulatory uncertainty in some jurisdictions
❌ No FDIC insurance
```

---

## Market Types

### Kalshi: Primarily Binary

```
Most Kalshi markets:
"Will [X] happen?" → Yes/No

Examples:
- "Will Fed raise rates in March?"
- "Will [movie] gross $100M opening?"
- "Will unemployment be below 4%?"
```

### Skepsis: Continuous

```
Skepsis markets:
"What will [X] be?" → Range selection

Examples:
- "Where will rates be after Fed meeting?"
- "What will [movie] opening gross be?"
- "What will unemployment be?"
```

**Key insight:** Kalshi's binary format loses information that Skepsis preserves.

## Example: Economic Prediction

### On Kalshi

```
Market: "Will December CPI be above 3.0%?"

Buy Yes at $0.65 → Win $1 if CPI > 3.0%
Buy No at $0.35 → Win $1 if CPI ≤ 3.0%

Information: 65% think CPI > 3.0%
Missing: What if CPI is 3.5%? 4.0%? 2.8%?
```

### On Skepsis

```
Market: "What will December CPI be?"

2.0-2.5%:  5%
2.5-3.0%:  25%
3.0-3.5%:  40%  ← Most likely
3.5-4.0%:  22%
4.0%+:     8%

Information: Full distribution
- Expected CPI: ~3.2%
- Most likely range: 3.0-3.5%
- Tail risk visible
```
